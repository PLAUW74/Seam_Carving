================================================================================
                        SEAM CARVING TOOL - HOW TO USE
================================================================================

This folder contains the pre-built application. No installation needed.

HOW TO USE
----------

1. Put your images in the "images" folder.

2. Double-click "SeamCarvingApp.bat".

3. Choose an option from the menu:
   - 1-3 (Standard Algorithms):
     - You will be asked for:
       - Input image name (e.g., input.jpg)
       - Output image name (e.g., input_carved.jpg)
       - Number of seams (e.g., 50)
       - Direction: "vertical" (to make narrower) or "horizontal" (to make shorter).
       - Visualize: 'y' to see every seam being removed (slow).

   - 4 (Interactive Tool):
     - You will be asked for:
       - Input image name (e.g., input.jpg)
     - A window will open.
     - Move the sliders to resize the image in real-time.
     - Press 's' to save your result as "interactive_result.jpg".
     - Press 'q' or ESC to close the window.

4. Check your results in the "images" folder.
