================================================================================
                        SEAM CARVING TOOL - HOW TO USE
================================================================================

HOW TO USE
----------

1. Put your images in the "images" folder

2. Double-click "SeamCarvingApp.bat"

3. Choose your algorithm:
   - Option 1: Dynamic Programming (Better quality, slower)
   - Option 2: Greedy Algorithm (Faster, lower quality)

4. Answer the prompts:
   - Input image name: e.g., input.jpg
   - Output image name: e.g., input_dp.jpg
   - Number of seams to remove: e.g., 50 
   - Direction: "vertical" or "horizontal"
     * vertical = make narrower (remove width)
     * horizontal = make shorter (remove height)

5. Check your results in the "images" folder:
   - Your carved image
   - comparison.jpg (shows before and after)
